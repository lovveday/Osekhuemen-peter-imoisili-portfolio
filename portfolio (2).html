About

I'm Osekhuemen Peter-Imoisili, a Data Analyst & Data Engineer who cares about the whole path data takes — from a raw, messy file to a decision someone actually makes because of it.

My work spans data cleaning, SQL modeling, pipeline automation, and BI dashboarding. I've built star-schema models from scratch, and I've also known when to walk away from one because the data didn't support it. I care about numbers being right before they're pretty.

When I'm not deep in a dataset, I'm asking "does this actually answer the business question?" — because a dashboard that looks good but says nothing isn't worth shipping.
